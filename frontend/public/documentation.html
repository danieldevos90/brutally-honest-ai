Quick Start

Brutally Honest AI is a voice validation platform that transcribes audio and provides unfiltered AI analysis with fact-checking capabilities.

1

Connect Device

Connect your ESP32S3 device via USB or Bluetooth from the Home page.

2

Upload Documents

Add reference documents to the Knowledge Base for fact-checking.

3

Record & Analyze

Record audio and get AI-powered transcription with brutal honesty analysis.

Features

Audio Recording

Record audio via ESP32S3 device with SD card storage. Support for USB and Bluetooth connections.

Whisper Transcription

High-quality audio-to-text conversion using OpenAI Whisper running locally on your device.

LLAMA Analysis

Local LLM processing with Ollama for sentiment analysis, summarization, and brutal honesty feedback.

Fact Checking

Cross-reference claims against your uploaded document knowledge base for validation.

Knowledge Base

Upload PDF, DOC, TXT files to build a searchable vector database for document retrieval.

Profile Management

Create profiles for clients, brands, and persons to track facts and validate claims.

Hardware Setup

Supported Devices

  • XIAO ESP32S3 Sense with Expansion Board
  • SD Card for recording storage
  • OLED Display (optional)

Connections

  • USB: Connect via USB-C cable
  • Bluetooth: Connect wirelessly (requires initial USB setup)

LED Indicators

  • Solid: Ready
  • Blinking: Recording in progress
  • Fast blink: Processing/uploading

‚ùì Troubleshooting

Device not detected?

Try putting the ESP32S3 in boot mode: Hold BOOT button, press RESET, release BOOT after 2 seconds.

Transcription slow?

Consider using the "base" Whisper model instead of "medium" for faster processing.

LLAMA not responding?

Check if Ollama is running: systemctl status ollama

Document upload failing?

Ensure file is under 10MB and in supported format (TXT, PDF, DOC, DOCX).
